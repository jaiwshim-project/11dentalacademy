1. 개요

가이드 엔진이란?

💡 가이드 엔진 소개

가이드 엔진(Guide Builder)은 코딩 없이 드래그 앤 드롭만으로 사용자 가이드, 매뉴얼, 튜토리얼 문서를 쉽게 만들 수 있는 도구입니다.

다양한 컴포넌트를 조합하여 전문적인 가이드 문서를 빠르게 제작하고, HTML 파일로 내보내어 어디서든 활용할 수 있습니다.

🚀 주요 특징

드래그 앤 드롭 / 8가지 컴포넌트 / 3가지 템플릿 / 8가지 테마 색상 / 실시간 미리보기 / HTML·JSON 내보내기

💻

2. 화면 구성

인터페이스 이해하기

📄 3단 레이아웃

가이드 엔진은 세 개의 영역으로 구성되어 있습니다.

영역 설명
왼쪽 사이드바 컴포넌트 목록과 템플릿이 있습니다. 여기서 원하는 요소를 드래그합니다.
중앙 캔버스 가이드가 만들어지는 편집 영역입니다. 컴포넌트를 여기에 드롭합니다.
오른쪽 패널 가이드 정보, 요소 속성, 테마 설정을 할 수 있습니다.

🔧 상단 툴바

상단에는 주요 기능 버튼들이 있습니다.

버튼 기능
🗑 초기화 캔버스의 모든 내용을 삭제합니다.
👀 미리보기 새 창에서 완성된 가이드를 미리 봅니다.
💾 저장 브라우저에 현재 상태를 저장합니다.
📥 내보내기 HTML 또는 JSON 파일로 내보냅니다.

4. 사용 방법

기본 워크플로우

🚀 가이드 만들기 5단계

1

컴포넌트 드래그

왼쪽 사이드바에서 원하는 컴포넌트를 마우스로 클릭한 채 끌어옵니다.

2

캔버스에 드롭

중앙 캔버스 영역에 마우스를 놓으면 컴포넌트가 추가됩니다.

3

내용 편집

추가된 요소의 연필(✎) 아이콘을 클릭하면 오른쪽 패널에서 내용을 수정할 수 있습니다.

4

순서 조정

화살표(▲▼) 버튼으로 요소의 순서를 위아래로 이동할 수 있습니다.

5

미리보기 & 내보내기

미리보기로 결과를 확인하고, 내보내기로 HTML 파일을 다운로드합니다.

✅ 빠른 시작 팁

처음 사용하신다면 왼쪽 하단의 템플릿을 클릭해 보세요. 기본 구조가 자동으로 추가되어 수정만 하면 됩니다!

📃

5. 템플릿

미리 만들어진 구조

🎨 3가지 템플릿

빠른 시작을 위한 템플릿이 제공됩니다.

템플릿 구성 용도
📃 기본 가이드 헤더 + 섹션 제목 + 단계 목록 + 팁 박스 일반적인 사용 가이드
🎓 튜토리얼 헤더 + 개요 + 텍스트 + 기능 그리드 + 팁 상세 학습 자료
❓ FAQ 헤더 + 질문/답변 반복 구조 자주 묻는 질문

⚠ 주의

템플릿을 불러오면 기존 작업 내용이 덮어씌워집니다. 중요한 작업은 미리 저장해 주세요.
